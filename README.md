# 使用方法
安装
```bash
$ npm i @ids-react/camel-ui
```

使用
```
import { Button } from '@ids-react/camel-ui';
import '@ids-react/camel-ui/dist/index.css';

<Button>按钮</Button>
```

参考`App.tsx`中每个组件的使用方法，基本都非常简单，每个组件1-3行代码。
